# Brave Search

Web search via Brave's Data-for-Search API. The free tier covers 2k queries/month.

## Get an API key

1. Sign up and create a key at https://brave.com/search/api/.
2. Pick the free plan (or a paid plan for higher limits). The key starts with `BSA`.

## Connect it to Gini

When Gini needs web search it shows a **Connect Brave Search** card in the chat — paste the key there and submit. You can also connect it from a skill's **Set up** button on the Skills page. Gini verifies the key against the live search endpoint before saving.

## Use it

Just ask Gini to search the web — for example, "search the web for the latest Bun release notes." Gini runs the query through Brave and works from the results.
