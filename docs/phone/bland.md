# Bland AI

Outbound AI phone calls via [Bland AI](https://bland.ai). Pay-per-minute, with free trial credits available.

## Get an API key

1. Sign up at https://app.bland.ai and create a key under **Settings → API Keys**.
2. The key starts with `org_`.

## Connect it to Gini

When Gini needs to place a call it shows a **Connect Bland AI** card in the chat — paste the key there and submit. You can also connect it from a skill's **Set up** button on the Skills page. Gini verifies the key against the live calls endpoint before saving.

## Use it

Just ask Gini to make a call — for example, "call the restaurant and book a table for two at 7pm." Gini confirms the number and what it will say, places the call, and reports back the summary and transcript.
